BishBosh is a simple number game where specific words are used in place of numbers based on their divisibility and the output is displayed based on the given total Loop count. In this game :

  • "Bish" is used when a number is divisible by the Bish number you choose(Example: 3).
  • "Bosh" is used when a number is divisible by the Bosh number you choose(Example: 4).
  • "Bish-Bosh" is used when a number is divisible by both the Bish and Bosh numbers.(Example: 12).
  • Sample Output: 1, 2, Bish, Bosh, 5, Bish, 7, Bosh, Bish, 10, 11, Bish-Bosh, 13, 14, Bish [...]
